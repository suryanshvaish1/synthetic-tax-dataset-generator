# Synthetic Tax Dataset Generator

##  Overview
This project generates synthetic tax datasets including financial documents and tax return forms using Python.

##  Features
- Generates 40+ synthetic tax datasets
- Creates PDF documents:
  - W-2
  - 1099-INT
  - 1099-DIV
  - Form 1040
  - Schedule B
  - State Tax
  - Summary Report
- Maintains structured folder organization
- Fully automated pipeline

## 🛠 Tech Stack
- Python
- Faker
- ReportLab
